The idea is to create a collective intelligence system (CIS) based on the metadata which is offered by wienbibliothek.

CIS Definition: “Harness the collective intelligence of connected groups of people by providing a web-based environment to share, distribute and retrieve topic-specific information” (Musil et al., 2015).

CIS design pattern used for this prototype: Comments (Reviews)

Problem: users cannot get in contact and engage with others for conversation about content

Solution: enable users to express their opinion and encourages discussion

Used in: YouTube, Yelp, Flicker, Facebook [1]

Description:

The service is a web application that allows users to comment (review) the books provided by wienbibliothek. The backend of the application retrieves the metadata from wienbibliothek. The interfacing is realized using the OAI-PMH access offered.

Definition:

Open Archives Initiative – Protocol for Metadata Harvesting

Combination of protocol and format for exchanging metadata between Libraries, Archives and Museums. Output = XML / Dublin Core Qualified

The user interface is a traditional web application which offers end users following features:

1. Browse the metadata from wienbibliothek

2. Search the metadata

3. Create new comment (reviews)

4. Browse existing comments (reviews)

5. User generated content is stored in a persistent database
